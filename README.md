# template-web

Vite + Vue 3 + VueRouter 4 + TailwindCSS

## Get Started

Install [degit](https://github.com/Rich-Harris/degit):

```
npm i -g degit
```

In the target folder,

```
degit yzITI/template-web
```

Change project name in  `package.json`
```
"name": "project-name"
```

Install dependency

```
npm i
```

Run dev server

```
npm run dev
```
